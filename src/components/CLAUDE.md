# Component Library Context: src/components

Guidelines for UI building blocks.

## Design tokens
- Use `Tailwind` classes.
- Standard gold color: `text-[#D4AF37]` or `border-[#D4AF37]`.

## Component Types
- **Shared**: Common buttons, inputs, modals.
- **Wine**: Specific components for wine display (Price tags, origin badges).

## Animations
- Use `framer-motion` for transitions (`initial`, `animate`, `hover`).
- Keep micro-animations subtle (0.2s - 0.5s duraton).
