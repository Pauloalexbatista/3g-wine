# Project History - 3GWine

## Phase 1: Foundation (Early 2026)
- **Objective**: Establish the luxury brand identity and basic storefront.
- **Key Milestones**:
    - Project initialization with Next.js 14 and Tailwind CSS.
    - Implementation of the dark/gold luxury design system.
    - Integration with Supabase for Auth and Database.
    - Creation of the Hero section with dynamic video background.
    - Development of the "A Nossa Seleção" grid.

## Phase 2: Web3 & Virtual Wines (March 2026)
- **Objective**: Launch the virtual wine (NFT) experience.
- **Key Milestones**:
    - Development of the `/vinhos-virtuais` landing page.
    - Implementation of 3D/hover effects for NFT cards.
    - Setup of basic detail pages for virtual wines.
    - Affirmation of the "misterioso e premium" aesthetic.

## Phase 3: Commercial Stability (In Progress)
- **Objective**: Finalize payment flows and perfect the UX/UI.
- **Focus**:
    - Integration with **IFTHENPAY** for local Portuguese payments.
    - Refinement of `/vinhos-virtuais/[id]` detail pages.
    - Verification of responsiveness across all devices.
    - Implementation of HAM (Hierarchical Agent Memory) for context stability.
